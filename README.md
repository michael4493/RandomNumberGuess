# Python Number Guessing Game 🎮

A command-line number guessing game built with Python. This project demonstrates foundational programming concepts including control flow, user input handling, and the use of built-in modules.

## ✨ Features
* **Customizable Range:** Users can define the minimum and maximum boundaries for the target number.
* **Attempt Tracking:** The game accurately counts and displays the total number of guesses taken to win.
* **Interactive Feedback:** Provides real-time "Bigger!" or "Smaller!" hints based on user input.

## 🛠️ Tech Stack
* **Language:** Python 3
* **Libraries:** `random` (Standard Library)
